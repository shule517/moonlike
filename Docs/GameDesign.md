# Moonlike - ゲームデザインドキュメント

## コンセプト

**「すべての行動に意味がある」**

ラブデリックの「moon」をオマージュした、ループ型アドベンチャーゲーム。
世界の終わりを何度も繰り返すことで、少しずつ世界が変化していく。
プレイヤーの小さな行動一つ一つが、次の世界へと引き継がれる。

---

## ゲームの基本構造

### ループサイクル

```
① 行動フェーズ（アドベンチャーパート）
    ↓
② 世界の終わり（太陽が膨張し、白い光に包まれる）
    ↓
③ 月の女王パート（行いの振り返り）
    ↓
① 次の周回へ（変化した世界で目覚める）
```

### 時間の流れ

- ゲーム内時間が経過すると、やがて太陽が膨張し世界が終わる
- プレイヤーの行動の有無に関わらず、時間は流れ続ける
- 「何もしない」という選択も、ひとつの行動として記録される

---

## 1周目の流れ

### シーン1: 目覚め（家の中）

**場所**: 主人公の家、ベッドの上

**演出**:
- 暗転から徐々に明るくなる
- 母親の声（画面外から）

**セリフ**:
```
「・・しょうた しょうた・・
起きなさい 朝よ
いつまで寝てるの？」
```

**プレイヤーのアクション**:
- ベッドから起き上がる
- 家の中を探索できる（アイテム：水）
- 外へ出る

### シーン2: 砂漠（外の世界）

**場所**: 家の外、見渡す限りの砂漠

**環境**:
- 四季がない、常に乾いた世界
- 大きな恐竜の骨が落ちている
- 太陽が空に見える

**オブジェクト**:
- 恐竜の骨（調べられる）
- 土（種を植えられる場所）
- 種（アイテムとして入手可能）

**プレイヤーのアクション**:
- 種を拾う
- 土に種を植える
- 水をあげる

### シーン3: 世界の終わり

**トリガー**: 一定時間経過後、または特定の行動後

**演出**:
1. 太陽が徐々に大きくなる
2. 画面が白く染まっていく
3. すべてが白い光に包まれる

### シーン4: 月の女王

**場所**: 虚空（黒い背景に月の光）

**月の女王のセリフ**:
```
「…時が来たようです。
この世界はただいま終わります。

けれど、あなたの行いは
消えてしまうわけではありません。

あなたは地に緑を植えました。

それはとても小さなことかもしれません。

ですが、そのひとつひとつが
次の世界へと静かに息づいていくのです。

あなたの人生は
たしかに短ったけれど、
意味のないことなんて何ひとつありません。

わたしは見ていました。
ずっと見守っていましたよ。」
```

---

## 2周目の変化

### 変化の内容

1周目で種を植えたことにより:
- **季節が生まれる**（春が訪れる）
- 砂漠だった世界が緑で覆われる
- 植えた種が大きく育っている
- 草が点在している

### 目覚めの変化

同じ母親の声で目覚めるが、外に出ると世界が一変している。

### 春のメッセージ

外に出ると以下のメッセージが表示される:
```
…世界が変わっている。
あの種から、緑が生まれた。
風が優しい。
春が来たんだ。
小さな行動が、
世界を変えたのかもしれない。
```

---

## プレイヤーの行動と世界の変化

| 行動 | 次の周回での変化 |
|------|------------------|
| 種を植える | 季節が生まれる |
| 何もしない | （今後の実装で意味を持たせる） |
| 恐竜の骨を調べる | （今後の実装） |

---

## 登場人物

### 主人公（しょうた）
- 名前のみ、姿は見せない（または後ろ姿のみ）
- プレイヤーの分身

### 母親
- 声のみで登場
- 毎回の目覚めの合図

### 月の女王
- 世界の終わりを見届ける存在
- プレイヤーの行動をすべて見守っている
- 優しく、肯定的なメッセージを伝える

---

## 世界観

### 設定
- 時間が崩壊しかけている世界
- 太陽が異常膨張を繰り返している
- 四季が失われ、砂漠化が進んでいる
- 過去には恐竜も生きていた

### テーマ
- **すべての行動の肯定**
- **小さな積み重ねが世界を変える**
- **終わりは新しい始まり**
